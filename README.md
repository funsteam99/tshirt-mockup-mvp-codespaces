# T‑Shirt Mockup MVP（拖拉框版）— Codespaces Ready

你要的「商用產品感」操作：上傳底圖（空白 T 恤商品照/平鋪照）＋上傳圖案 → 在畫面上直接「拖拉 / 縮放 / 旋轉」圖案 → 生成 Mockup。

- ✅ 手機也可用（拖拉/縮放/旋轉）
- ✅ 不用點四點
- ✅ 後端仍用「透視變形」合成（穩、可控）

---

## Codespaces 啟動
```bash
./start-dev.sh
```

打開 Ports 的 `5173`（Web）。

---

## 操作
1) 上傳底圖  
2) 上傳圖案  
3) 在畫面上拖曳圖案到胸口，拉角落縮放，旋轉把手旋轉  
4) 按「生成 Mockup」

> 小技巧：用手指點一下圖案會出現控制框（Transformer）。
